NFL Multivariate Analysis

Introduction: This project is a collection of homework assignments for the college level course “Multivariate Analysis” at Rutgers University. The data used for all the homework assignments was self-created NFL Data Set from various online sources and used to track weekly performance for each team in each one of their matchups. There are 14 homework assignments in total that are presented as one project. The following statistical topics are covered in the project: Finding Mean Vectors Finding Variance-Covariance Matricies Finding Correlation Matricies Assessing Bivariate Normality in Pairs with Several Statistical Tests for Normality Checking Data Set for Outliers & Removing Them Creating Bonferroni Confidence Intervals Testing Given Numbers Supposedly Representing Means & Testing Them Against Actual Population Means Testing Mean Difference in Two Different Samples Creating Multivariate Linear Models Conducting Principal Component Analysis (PCA) Obtaining Eigenvalue & Eigenvector for PCA Model Selection for Mulitvariate Models Creating & Analyzing Growth Curves Conducting Explanatory Factor Analysis Performing Maximum-Likelihood Factor Analysis Covariance Matrix W/ Loadings Comparing Effectiveness of Various Factor Analysis Computing Canonical Correlations & Testing for Signifigance Generating Helio Plots Calculating Euclidean Distances Creating Dendrograms Conducting K-Means Clustering Note: Some reflections and answers were required for some of the assignments and their subsections, BUT not for all.
